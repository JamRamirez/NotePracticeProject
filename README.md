# Note Practice Project

A simple command-line note management application built with Go as part of my learning journey into backend development.

## Features

- Create notes from user input
- Display note information
- Save notes as JSON files
- Basic input validation
- Error handling
- Structs and methods
- File handling

## Technologies

- Go

## Purpose

This project was created to practice fundamental Go concepts while becoming more comfortable with backend development. The focus is on writing clean, readable code and understanding how Go handles data, methods, pointers, and JSON serialization.
